## 🌟 Overview
**jobZee** is a modern full-stack **job portal application** built with the MERN stack (MongoDB, Express.js, React, Node.js).  
It allows **job seekers** to browse and apply for jobs, while **recruiters** can post new job listings.  

💡 Designed for simplicity, speed, and scalability, jobZee provides a seamless experience for all users.  

---

## ✅ Features
- 🔐 **User Authentication** – Sign up, login, and logout  
- 💼 **Job Posting** – Recruiters can post new jobs  
- 🔍 **Job Search & Filtering** – Search by title, location, skills  
- 📝 **Job Applications** – Apply for jobs with a single click  
- ☁️ **Cloud Integration** – Upload logos or images using Cloudinary  
- 📱 **Responsive Design** – Works on desktop, tablet, and mobile  
- 🔑 **JWT Authorization** – Secure access to protected routes  

---

## 🛠 Technologies Used

### Frontend
- ⚛️ React.js (Vite)  
- 🧭 React Router  

### Backend
- 🟢 Node.js  
- 🚂 Express.js  
- 🍃 MongoDB & Mongoose  
- ☁️ Cloudinary (for image uploads)  
- 🔐 JWT for authentication  

### Dev Tools
- 🔄 Nodemon (auto-reload server)  
- 📬 Postman (API testing)  
- 🐙 Git & GitHub
