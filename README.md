# 🎯 jobZee – MERN Stack Job Portal

jobZee is a full-stack Job Portal Web Application built using the MERN Stack (MongoDB, Express, React, Node.js).  
It allows Employers to post and manage jobs, while Job Seekers can browse and apply for openings.

---

## 🚀 Features

### 👤 Authentication
- User Login & Registration  
- Role-based access: Employer / Job Seeker  
- Secure JWT authentication (HTTP-only cookies)

### 🧑‍💼 Employer Features
- Post new jobs  
- Edit or delete jobs  
- View job applications  

### 👨‍💻 Job Seeker Features
- Browse all job listings  
- Search & filter jobs  
- Apply for jobs  
- View applied jobs  

### 💰 Salary Options
- Fixed Salary  
- Salary Range (From → To)

---

## 🛠 Tech Stack

| Layer       | Technology |

| Backend     | Node.js, Express.js |
| Database    | MongoDB, Mongoose |
